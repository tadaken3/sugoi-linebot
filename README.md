# sugoi-linebot

運動を継続するための「運動報告部」というメソッドを知りました。
こちらを実践するためにLINEボットを作成しました。

詳細は以下をご覧ください。<br>
http://tadaken3.hatenablog.jp/entry/sugoi-linebot

### 使い方

①LINEボットを友だち登録する<br>
②運動したら「ジム」もしくは「走った」と報告する<br>
③ボットが「すごい」と褒めてくれる<br>
